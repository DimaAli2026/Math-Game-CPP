# 🧮 Math Game Project

A console-based mental calculation challenge developed in **C++**. This project focuses on logic, struct usage, enums, and dynamic user interaction.

## 🚀 Features
- **Dynamic Levels:** Choose between Easy, Med, Hard, and Mix.
- **Operation Types:** Addition, Subtraction, Multiplication, Division, or Mix.
- **Real-time Feedback:** Instantly tells you if your answer is correct.
- **Interactive UI:** Changes screen color based on your performance (Green for win, Red for loss).
- **Game Summary:** Detailed report at the end showing your score and level.

## 🛠️ Concepts Used
- **Enums & Structs:** For organized and clean code.
- **Randomization:** Using `srand()` and `rand()` for unpredictable questions.
- **Control Flow:** Advanced use of `switch` cases and `do-while` loops.
- **Functional Programming:** Modular code with reusable functions.
